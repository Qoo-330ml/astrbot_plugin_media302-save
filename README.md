# media302转存插件
> 一个基于 [AstrBot](https://github.com/astrxbot/astr-bot) 的插件。
> 本插件用于通过media302软件转存115网盘分享链接

## 配置参数
- media302_url：media302软件服务地址 (如: https://302.abc.xyz:666)
- folder：默认转存目录
- media302_token：media302软件认证token

## 使用指令
- /mediasave [115分享链接]  转存指定的115分享内容

## 注意事项
1. 请确保media302服务正常运行
2. token需具有有效权限
3. 分享链接需完整包含密码部分
